## Vision

Combine the strengths of the Kanseki Repository design with interactive features from HXWD.org into a new application for both online and offline use. Ideally, most features should be usable locally without active network.  Local work should be syncable to a common workspace.

GitHub will play a central role here as a storage backend for all the common data (@krp-yaml) and user data, which will be accessed in the account of the user.  

The meetings will be used to accompany the development, discuss strategies for resolving problems, new directions and related questions.
