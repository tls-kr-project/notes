# Frontend 

## Layout
- we will have a SPA with a flexible VS-Code like layout:
  - menu bar, 
  - main panel, consisting of
    - hideable left and side panels
        - toc, search results, etc. can be displayed here, optionally in vertically stacked subpanels
    - central area: This is the main working area  (might house multiple sub-panels)

  - quick access buttons in a dock (left most)

  - status bar, 

- In the menu bar, there are sections to access other parts of the app, a search bar and a login button, very similar to the current HXWD.org
