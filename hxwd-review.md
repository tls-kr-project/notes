# Lessons learned from the HXWD.org project

This document mainly lists problems that 

## Data format 
- Texts and 'dictionary' data are kept separate
- For texts, the centrality of the phrase, which makes it impossible to correct obvious errors is a design mistake
- The concept files are difficult to maintain in the current format, edits produce errors
- Intertextuality not actualized

## Functionality
- displaying a page is way to slow
- advanced search not possible or 
- no true kwic display (not reverse sortable by prev. chars)
